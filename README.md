# Moje portfolio

Proste portfolio z panelem CMS — żeby rekruterzy mogli szybko zobaczyć moje projekty w jednym miejscu.

## Stack

- Next.js
- React
- Material UI
- Prisma + Neon (PostgreSQL)
- iron-session (logowanie do panelu admina)

### etc

- Logowanie: `/admin/login`
- Dodawanie projektów: `/admin/projects` (po zalogowaniu)
